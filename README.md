# Weather App
A simple weather app that searches for current weather conditions in Czech cities using OpenWeatherMap API.

## Features
- Search for any city
- Display current weather data
- API integration
- Responsive design

## How to Use
Enter a city name and click Search to see current weather information.
The API isn't working right now due to security concerns.

## Tech Stack
- HTML
- CSS
- JavaScript
- REST API
