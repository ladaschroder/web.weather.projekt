# Weather App

A simple weather app that searches for current weather conditions in Czech cities using OpenWeatherMap API.

## Features

- City search by name
- Displays temperature (in Celsius), description, and weather icon
- Error handling for invalid inputs

## How to Use

Enter a city name and click Search to see current weather information.

## Files

- `weather.html` - HTML structure
- `style.css` - Styling and layout
- `script.js` - Weather data fetching logic

## Technical Stack

HTML5, CSS3, JavaScript (async/await, Fetch API)
